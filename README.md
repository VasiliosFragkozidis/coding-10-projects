# coding-10-projects
Repository, to push , pull and clone different projects created by the members of Coding Factory 10.  Be creative! 


## Οδηγίες συμμετοχής

### Πρώτη φορά:
git clone https://github.com/[coding-10-projects].git
cd [coding-10-projects]
git checkout -b member-[username_σου]
git push origin member-[username_σου]

### Κάθε φορά που δουλεύεις:
git checkout member-[username_σου]   # πήγαινε στο branch σου
git add .
git commit -m "Περιγραφή αλλαγής"
git push origin member-[username_σου]

### Κανόνες:
- ✅ Δούλευε ΜΟΝΟ στο δικό σου branch
- ❌ Μην κάνεις push στο main
- 📩 Για να μπει κώδικας στο main → άνοιξε Pull Request
